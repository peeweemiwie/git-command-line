# Terminal command line


## Create a directory

    $ mkdir directory-name
    
## Delete a directory 

    $ rmdir directory-name
    
## Delete a directory with file(s)

    $ rm -rf directory-name
    
## Create a few file

    $ touch new-file


## Rename a file

    mv oldfilename newfilename
    
## Check what is inside the directory

    ls 
    ls -la
    ls -hla
